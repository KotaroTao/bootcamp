# 歯科医院向け Claude Code 研修プログラム

## 「AIで歯科医院の業務が変わる」― スタッフ全員がAI人材になる15日間

## コンセプト

「プログラミング経験ゼロでも、自分の手でアプリが作れる」

Claude Codeは、自然な日本語で指示するだけでプログラムを自動生成するAIツールです。
本研修では、歯科医院の現場で「本当に使えるもの」を自分たちの手で作り上げる体験を通じて、
AI時代に必要なスキルを身につけます。

## プログラム構成

### 第1部：導入コース（3時間 × 5日間）

**ゴール：全員がClaude Codeを使いこなし、簡単なアプリを一人で作れる状態にする**

| 日程 | テーマ | 内容 |
|------|--------|------|
| **Day 1** | はじめの一歩 | Claude Codeとは何か？／AIに仕事をさせるという考え方／WEB版の環境構築（全員が動く状態まで完全サポート）／最初の「Hello World」体験 |
| **Day 2** | AIへの指示の出し方 | プロンプトの基本（具体的に・段階的に・例を示す）／うまくいく指示 vs うまくいかない指示の違い／ハンズオン：簡単な計算ツールを日本語だけで作る |
| **Day 3** | Webアプリの仕組み | HTML・CSS・JavaScriptをざっくり理解（書けなくてOK）／Claude Codeに「見た目」を指示するコツ／ハンズオン：歯科医院の受付用カウントダウンタイマーを作成 |
| **Day 4** | データを扱うアプリ | データの保存・表示の仕組み／Claude Codeでフォームやリストを作る／ハンズオン：院内の備品チェックリストアプリを作成 |
| **Day 5** | ミニアプリ発表会 | 各自が「自分の医院で使いたいもの」を企画→Claude Codeで制作→発表／振り返りと質疑応答 |

---

### 第2部：実践コース（3時間 × 5日間）

**ゴール：歯科医院の実務で使える本格的なアプリを開発し、実際に現場で運用を開始する**

| 日程 | テーマ | 内容 |
|------|--------|------|
| **Day 6** | 実践の準備 | 導入コースの復習と応用テクニック／より複雑な指示の出し方（要件定義の考え方）／チーム開発の進め方 |
| **Day 7** | 患者向けアプリ開発 | 例：治療説明用ビジュアルツール／術後ケアガイド（患者さんに見せるアプリ）／ハンズオン：自院の患者説明ツールを制作 |
| **Day 8** | 業務効率化アプリ開発 | 例：在庫管理システム／スタッフシフト管理ツール／日報・申し送りアプリ／ハンズオン：自院の課題を解決するアプリを制作 |
| **Day 9** | アプリの仕上げと改善 | デザインの改善（見やすく・使いやすく）／エラー対応の方法／スマホ対応／テストと修正のサイクル |
| **Day 10** | 最終発表＆運用開始 | 完成アプリのプレゼンテーション／実際の業務への導入計画作成／今後の自走に向けたTips／修了証授与 |

## 研修で作れるアプリ例

歯科医院の現場ですぐに使えるものを、受講者自身が制作します。

- **患者説明ツール** ― 治療内容をイラストやアニメーションで分かりやすく説明
- **問診票アプリ** ― タブレットで患者さんが直接入力、紙の手間を削減
- **在庫管理アプリ** ― 材料の在庫をスマホでチェック、発注時期を自動通知
- **スタッフ申し送りアプリ** ― シフト交代時の情報共有をデジタル化
- **予約リマインダー** ― 患者さんへの予約確認を自動化
- **院内マニュアル検索** ― 新人スタッフが手順をすぐに検索できるツール
- **自費診療の料金シミュレーター** ― 患者さんに治療プランと費用を視覚的に提示
