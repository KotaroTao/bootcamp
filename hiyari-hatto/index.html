ヒヤリハット報告アプリ

歯科医院のヒヤリハット報告を
かんたんに記録・共有できるアプリです

まず、使い方を選んでください：

セットアップ

順番に進めるだけでOK！

Step 1: Google スプレッドシートを作る

  1. 下のボタンで Google スプレッドシートを開く
  2. 左上の 「空白のスプレッドシート」（＋マーク）をクリック
  3. 名前を 「ヒヤリハット報告」 などに変更
Google スプレッドシートを開く ↗

Step 2: Apps Script を開く

作ったスプレッドシートの画面で：

拡張機能 → Apps Script

新しいタブが開いて、コード編集画面が表示されます。

Step 3: コードを貼り付ける

  1. Apps Scriptの画面で、最初のコードを全選択して削除
  2. 下の「コードをコピー」ボタンを押す
  3. Apps Script に貼り付け（Ctrl+V）
  4. 保存（Ctrl+S）

Step 4: 初期設定を実行する

  1. Apps Script 上部の関数選択で 「initSheet」 を選ぶ
  2. 「実行」ボタン（▶） をクリック
  3. 権限の承認画面が出たら、許可する

成功すると、スプレッドシートに「報告データ」シートが追加されます。

Step 5: ウェブアプリとして公開する

デプロイ → 新しいデプロイ
  1. 歯車アイコン → 「ウェブアプリ」 を選択
  2. アクセスできるユーザーを 「全員」 に変更
  3. 「デプロイ」 をクリック
  4. 表示された URLをコピー

Step 6: URLを貼り付けて接続する

Step 5でコピーしたURLを貼り付けて「接続する」を押してください。

✔️

セットアップ完了！

さっそくヒヤリハット報告を始めましょう

報告データはGoogleスプレッドシートに自動保存されます。

1
Google スプレッドシートを作る 新しいスプレッドシートを1つ作るだけ
  1. 下のボタンで Google スプレッドシート を開きます
  2. 左上の 「空白のスプレッドシート」（＋マーク）をクリック
  3. 名前を 「ヒヤリハット報告」 などに変更
    （左上の「無題のスプレッドシート」をクリックすると変更できます）
Google スプレッドシートを開く ↗
Googleアカウントにログインしている必要があります。
医院で使う場合は、医院用のGoogleアカウントで作ることをおすすめします。
2
Apps Script を開く スプレッドシートの中から開けます

作ったスプレッドシートの画面で：

画面上部のメニューから
拡張機能 → Apps Script
をクリック

新しいタブが開いて、コード編集画面が表示されます。

最初から function myFunction() というコードが書いてありますが、このあと全部書き換えるので気にしなくてOKです。
3
コードを貼り付ける 下のコードをコピーして貼り付けるだけ
  1. Apps Script の画面で、最初から書いてあるコードを全部選択して削除
    （Ctrl+A → Delete、またはMacなら Cmd+A → Delete）
  2. 下の 「コードをコピー」 ボタンを押す
  3. Apps Script の画面に 貼り付け（Ctrl+V または Cmd+V）
  4. 左上の 保存ボタン（💾） をクリック
    （または Ctrl+S / Cmd+S）
4
初期設定を実行する ボタンを1回押すだけ
  1. Apps Script 画面の上部にある関数選択（「myFunction」と書いてある部分）をクリック
  2. リストから 「initSheet」 を選ぶ
  3. 左の 「実行」ボタン（▶） をクリック
⚠ 初回は「権限の承認」画面が出ます。以下の手順で許可してください：

1. 「権限を確認」をクリック
2. 自分の Google アカウントを選択
3. 「詳細」をクリック
4. 「〇〇（安全ではないページ）に移動」をクリック
5. 「許可」をクリック
成功すると、スプレッドシートに 「報告データ」 というシートが追加され、ヘッダー行が自動で作られます。 スプレッドシートのタブを確認してみてください。
5
ウェブアプリとして公開する 外部からアクセスできるようにします

Apps Script の画面で：

画面右上の
デプロイ → 新しいデプロイ
をクリック
  1. 左上の歯車アイコン（⚙）をクリック →「ウェブアプリ」を選択
  2. 設定を以下のようにする：
    • ☑ 説明：ヒヤリハット報告（なんでもOK）
    • ☑ 実行するユーザー：自分
    • ☑ アクセスできるユーザー：全員 ← ここが重要！
  3. 「デプロイ」 ボタンをクリック
  4. 表示された URL をコピー（📋アイコンでコピーできます）
URLは https://script.google.com/macros/s/xxxxx/exec という形式です。
このURLを次のステップで使います。
6
URLを貼り付けて接続する これで完了です！

Step 5 でコピーしたURLを下に貼り付けて、「接続する」を押してください。

スマホからアクセスできるようにする GitHub Pages（無料）で公開する方法
GitHub Pages を使うと、このアプリにスマホやタブレットからURLでアクセスできるようになります。完全無料です。

A. GitHub でリポジトリを作る

  1. github.com でアカウントを作成（無料・既にあればログイン）
  2. 右上の 「+」 → 「New repository」 をクリック
  3. 設定：
    • ☑ Repository name: hiyari-hatto（好きな名前でOK）
    • ☑ Public を選択
    • ☑ 「Add a README file」にチェック
  4. 「Create repository」 をクリック

B. ファイルをアップロードする

  1. 作成したリポジトリのページで 「Add file」 → 「Upload files」 をクリック
  2. 以下の3つのファイルをドラッグ&ドロップ：
    • index.html（このアプリ本体）
    • manifest.json（スマホ対応用）
    • sw.js（オフライン対応用）
  3. 下の 「Commit changes」 をクリック

C. GitHub Pages を有効にする

  1. リポジトリのページで 「Settings」（⚙ 歯車アイコン）をクリック
  2. 左のメニューから 「Pages」 をクリック
  3. 「Branch」のところで 「main」 を選択 → 「Save」
  4. 1〜2分待つと、上部に公開URLが表示されます
公開URLの例：
https://あなたのユーザー名.github.io/hiyari-hatto/

このURLをスマホのブラウザで開けば、すぐに使えます！
⚠ 注意： リポジトリは「Public（公開）」にする必要があります。
HTMLのソースコードは見られますが、報告データ（スプレッドシート）は見られないので安全です。
🏠
ホーム画面に追加する（アプリ風に使う） スマホのホーム画面にアイコンを置けます
ホーム画面に追加すると、普通のアプリと同じようにアイコンから起動できます。アドレスバーも消えてスッキリ。

iPhone（Safari）の場合

  1. Safari でアプリのURLを開く
  2. 画面下の 共有ボタン（▲が付いた四角いアイコン）をタップ
  3. 「ホーム画面に追加」 をタップ
  4. 名前を確認して 「追加」 をタップ

Android（Chrome）の場合

  1. Chrome でアプリのURLを開く
  2. 右上の 「⋮」（3点メニュー） をタップ
  3. 「ホーム画面に追加」 または 「アプリをインストール」 をタップ
  4. 「追加」 をタップ
追加すると、ホーム画面に「ヒヤリハット」のアイコンが表示されます。
タップするだけで、アドレスバーなしの全画面でアプリが開きます。
